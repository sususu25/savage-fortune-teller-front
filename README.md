# Savage Fortune Teller Front

React frontend for the Savage Fortune Teller birth chart roast.

## Setup

```bash
pnpm install
copy .env.example .env
pnpm dev
```

The frontend expects the backend at:

```env
VITE_API_BASE_URL=http://localhost:8000/api/v1
```

The app includes a demo reading, so the visual flow can be reviewed before the backend is running.
